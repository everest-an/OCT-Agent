# OCT-Agent Desktop 安装与升级说明

本说明面向普通用户，不需要了解命令行、Node.js、npm 或配置文件。

## 1. Windows 安装

1. 下载 `OCT-Agent Setup x.x.x.exe`。
2. 双击安装包，按提示完成安装。
3. 安装完成后，直接打开 OCT-Agent。
4. 第一次启动时，应用会自动完成环境检查和必要组件准备。
5. 几分钟后进入主界面，即可开始使用。

当前版本的首次启动通常会自动处理以下内容：

- 检查本机是否已有运行环境。
- 检查是否已安装 OpenClaw。
- 准备 Awareness Memory 所需组件。
- 启动本地服务。
- 如果你之前已经配置过模型，应用会尽量复用原来的设置。

如果网络较慢，首次启动时间会更长。安装过程中不需要手动打开终端，也不需要额外输入命令。

## 2. macOS 安装

1. 下载 `AwarenessClaw.dmg` 或 `OCT-Agent.zip`。
2. 按提示拖到“应用程序”文件夹，或直接解压后打开。
3. 第一次启动时，如果系统提示来源验证，请在系统允许后继续打开。
4. 应用会自动完成环境检查和必要组件准备。
5. 完成后进入主界面使用。

macOS 的首次启动逻辑与 Windows 基本一致，都会尽量自动完成准备工作。

## 3. 第一次使用时会看到什么

第一次进入应用时，会看到一个安装向导。

它会按顺序帮你完成：

- 检查环境
- 准备运行环境
- 准备 OpenClaw
- 准备记忆组件
- 启动本地服务
- 选择模型
- 选择记忆模式

如果你的电脑上已经有可复用的配置，应用会自动跳过一部分步骤，尽量减少重复操作。

## 4. 升级怎么做

当检测到新版本时，应用顶部会出现升级提醒。

你可以选择：

- 立即升级
- 稍后提醒
- 不再提醒

点击“立即升级”后，应用会自动完成升级检查和升级操作。升级完成后，建议：

- 重新打开应用
- 新开一个聊天会话

这样可以确保新配置和新组件完全生效。

## 5. 网络与离线说明

当前版本的设计目标是尽量减少用户手动操作。

- 如果网络正常，应用会自动在线准备和更新组件。
- 如果安装包内带有本地离线组件，应用会优先使用本地组件完成启动。
- 如果某些组件只能在线获取，而当前网络不可用，应用会提示稍后重试，而不是要求用户自己输入命令。

## 6. 遇到问题时建议怎么做

如果安装或升级没有顺利完成，建议按下面顺序尝试：

1. 关闭应用后重新打开。
2. 确认网络连接正常。
3. 再次点击升级或重新运行安装包。
4. 如果仍然失败，把错误提示截图发给支持人员。

普通用户不需要手动安装额外工具，也不需要自己配置镜像源。

## 7. 关于配置与安全

- 模型配置、插件配置和本地设置会尽量自动保存。
- 如果你导出配置文件，请妥善保管，因为它可能包含你的服务连接信息。
- 如果你使用的是共享电脑，不建议把敏感密钥长期保存在设备上。
